WebMEG: Browsing time series data using JavaScript

What is WebMEG?

WebMEG is a dynamic client side application developed primarily in JavaScript for viewing and analyzing time series data. Users can not only view but also interact with data on a web browser like never before. It does not need any outgoing data transfer over the network thus protecting data privacy.

How do I get started?

Download sample file and see WebMEG in action. Refer FAQ below for learning how to interact with the data

On what technologies is WebMEG built on?

WebMEG is developed mainly in JavaScript, some HTML and CSS. WebMEG uses Backbone for providing a solid model-view structure. As more data formats are added, the application as it grows can become big and therefore RequireJS is used for managing dependencies. It uses JQuery for DOM manipulation and Bootstrap for UI design. Since the amount of data to be visualized can be huge, WebMEG uses Dygraphs for plotting purposes. Dygraph is capable of handling millions of data points without getting bogged down and also provides flexibility in developing such a novel application.

How do I interact with the data?

Users can interact with their data in multiple ways. Either using buttons provided at the footer or using keyboard shortcuts
Key/Button Function
Up/Down Key Increase/Decrease amplitude of your data
Left/Right Key Scroll left/right of your data
Mouse click on series Highlight the series
Delete Mark highlighted series as bad and remove them from display
Left mouse click and drag Zoom into your data
Shift Key + drag Pan your data
Double mouse click Reset Zoom
Also user can use the menu bar to change the visibility of channels, mark channels as good or bad, create a selection to view only those channels of interest, edit selection and delete selection.

On what devices is it supported?

Currently, WebMEG is supported on all major browsers. Work is in progress to enable it across multiple devices like tablets, phones etc
